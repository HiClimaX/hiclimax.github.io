The Problem

Cities need climate projections. Current tools cannot deliver them.

Global climate models operate at 25–100 km resolution. Cities occupy 1–10 km. In between lies a fragmented landscape of isolated methods — each applied alone, each generating uncertainty it cannot quantify.

Fragmentation
Methods applied in isolation — no comparison, no benchmark, no common framework
Hidden Uncertainty
Single-method studies give false confidence — methodological uncertainty is never quantified
Inaccessibility
Workflows are undocumented, unreproducible, and not transferable to new cities
The Solution

One framework. Six methods. Transparent ensembles.

HiClimaX integrates all major downscaling families into a standardised, open-source pipeline. Outputs from every method are directly comparable. Ensemble products make uncertainty explicit and actionable.

Standardised I/O across all six method families
Built-in ensemble statistics and storyline generation
Native support for urban morphology, UHI, and anthropogenic heat
Reproducible from raw GCM to city-scale output — fully scripted
CORDEX-Urban aligned; WMO GFCS partnership pathway
Six Methods · One Framework

Every major downscaling family, integrated

Each module is an independent repository with standardised interfaces — plug one in, run all six, or combine any subset into an ensemble.

Direct Dynamical (D-DS)

Physically consistent kilometre-scale simulations using WRF or RegCM with urban canopy parameterisation. The gold standard for process fidelity.

WRF-UCM RegCM-urban CMIP6

Pseudo-Global Warming (PGW-DS)

Embeds GCM-derived warming signals into reanalysis-driven extreme events. Ideal for event attribution and stress-testing cities against future extremes.

Event-based Attribution

Statistical–Dynamical (SD-DS)

Combines circulation typing (weather patterns, k-means) with empirical regression. Computationally efficient; suitable for large multi-city ensembles.

Weather Typing Ensemble-ready

Land-Surface Physics (LSP-DS)

Offline land surface models (HRLDAS, Noah-MP) resolve surface fluxes and urban heat exchange at <1 km resolution — efficient yet physically grounded.

HRLDAS UHI analysis

Statistical (SDS)

Quantile mapping, delta-change, and regression-based methods for rapid multi-decadal projections across long time series with minimal computational cost.

Q-mapping Long time series

AI-Based (AI-DS)

CNN, GAN, and transformer architectures learn nonlinear scale-to-scale mappings. Orders of magnitude faster than RCMs once trained; scalable to any city.

PyTorch CNN / GAN Transformers
Workflow

From raw GCM to decision-ready ensemble in four steps

1

Ingest

Automatic download of ERA5, CMIP6, IMERG, and urban parameters via unified data interface

2

Downscale

Run one or all six modules independently on standardised inputs; each produces CF-compliant NetCDF

3

Evaluate

Automated skill scoring against observations — bias, RMSE, Taylor skill, extreme indices — via hiclimax-eval

4

Ensemble

Combine methods into probability distributions and storylines; export for impact models or APIs

# Install
pip install hiclimax

# Configure your city
from hiclimax import Framework, City

city = City("Tokyo", lat=35.7, lon=139.7)
fw = Framework(city, period="1991-2020")

# Run whichever modules you need
fw.run(modules=["D-DS", "PGW", "AI-DS"])

# Evaluate and build ensemble
results = fw.evaluate()
ensemble = fw.ensemble()
ensemble.storyline(scenario="SSP5-8.5")
